I want a web app that plans a road-trip holiday given a list of destinations, how long to spend at each destination, the desirability of each destination, how long a holiday it can be, how many hours of travel each day are allowed, a start/end point. Vehicle fuel consumption and fuel tank size

It will:
- map out a total route showing each days travel including a map and list. 
- the list will have details about that days travel including points of interest passed.
- the end point of each day that does not have a end point as a specified destination will have to find a place to stay with a strong preference for free-camp locations while loosely satisfying the number of hours travelled in the day (can go short or long)
- publish to a github page
